Proyecto de traducción al español de pokemon Fire Red v1.0 
  
Dependencias: 
- armips  
- cygwin  
- devkitARM  
  
Para compilar simplemente nombra tu rom como "rom.gba" y elige el offset  
donde quieras insertar los datos en el archivo "main.s". Luego ejecuta 
"make"
